1. 创建版本库
	1.1 创建一个空目录
		$ mkdir learngit	%创建一个文件夹
		$ cd learngit		%进入某个文件夹（或用于切换进入某个文件夹，地址开头是/c/Users/...）
		$ pwd			%显示当前目录
	1.2 创建Git可以管理的仓库
		$ git init			%初始化一个Git仓库
	1.3 把文件添加到仓库
		$ git add readme.txt		%把文件添加到仓库
		$ git commit -m "wrote a readme file" 	%将文件提交到仓库
2. 全局配置Git所使用的邮箱和昵称
	git config --global user.email "song971222@163.com"	%配置全局提交邮箱
